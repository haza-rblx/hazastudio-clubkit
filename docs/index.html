Quick Start

Tutorial langkah demi langkah — dari terima file sampai siap publish. Centang progress di kartu Deploy checklist di kanan saat kamu menyelesaikan tiap poin.

1

Terima file & buka Roblox Studio

Kamu akan menerima ClubKit.rbxm dari Hazastudio. File ini sudah berisi GUI, leaderboard boards, tools, dan semua script engine.

  1. Buka Roblox Studio → buka place venue kamu (atau buat place baru).
  2. Menu Home → Insert from File → pilih file .rbxm.
  3. Tunggu sampai folder kit muncul di Explorer.
✓ Berhasil jika: di Explorer terlihat minimal: ReplicatedStorage/Hazastudio_ClubKit, ReplicatedStorage/Hazastudio_ClubKitConfig, ServerScriptService/Hazastudio_ClubKit/Server, StarterPlayer/StarterPlayerScripts/Hazastudio_ClubKit, StarterGui (folder GUI 01- … 15-), ServerStorage/Tools, dan board leaderboard di Workspace.
2

Isi Group & Owner

Ini langkah pertama wajib — tanpa ini banyak fitur (owner command, community board) tidak jalan.

  1. Explorer → ReplicatedStorage → Hazastudio_ClubKitConfig → double-click ClubKitConfig (ModuleScript).
  2. Isi Group.GroupId dengan ID group Roblox venue kamu.
  3. Isi Group.OwnerUserId dengan UserId owner venue.
  4. (Opsional) tambahkan admin di AdminUserIds.
ClubKitConfig.Group = {
    GroupId = 12345678,
    OwnerUserId = 987654321,
}
✓ Berhasil jika: Play test tidak warn GROUP_ID tidak valid. Owner bisa pakai command owner (/showcase status).
Detail Group →
3

Matikan mode demo (showcase)

Kit mungkin masih punya file showcase untuk trailer. Production harus live, bukan data palsu.

  1. Explorer → ReplicatedStorage → Hazastudio_ClubKit → Shared → Config → cari ClubKitShowcase (ModuleScript).
  2. Production: hapus file, atau set ACTIVE = false.
  3. Play test → ketik /showcase status sebagai owner → harus OFF.
✓ Berhasil jika: leaderboard menampilkan data live / pesan kosong — bukan profil demo acak.
Detail Showcase →
4

Setup donasi cash (jika dipakai)

Lewati langkah ini jika venue tidak pakai donasi IDR. Kalau pakai Bagi-Bagi atau Saweria:

  1. Explorer → ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig → section Donation: set Provider = "bagibagi" atau "saweria".
  2. Isi ProviderLink dan ApiUrl di file yang sama.
  3. Explorer → ServerScriptService/Hazastudio_ClubKitSecrets/Secrets → isi DonationApiSecret (harus sama dengan worker).
✓ Berhasil jika: board cash tidak tampil "API not configured". Donasi test masuk (lihat Donation).
Detail Donation →Secrets →
5

Setup shop & donasi Robux

Membership (VIP/VVIP/Supreme) dan donasi Robux pakai Developer Products terpisah.

  1. Creator Dashboard → Monetization → Developer Products → buat product untuk tiap tier shop (Buy + Gift).
  2. Salin Product ID → paste ke ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig → Shop.Products.Tier1.BuyId / GiftId (ulangi per tier).
  3. Buat satu Developer Product untuk broadcast berbayar (pesan ke seluruh server via ikon topbar Broadcast).
  4. Salin ID product tersebut → ClubKitConfig.PaidBroadcast.ProductId (bukan ke Shop.Products).
  5. Buat Developer Products donasi Robux (atau biarkan server auto-load).
  6. Pastikan Features.Shop dan Features.DonationRobux = true jika dipakai.
✓ Berhasil jika: prompt shop muncul di game, ikon Broadcast menampilkan harga Robux, tidak error "Products Not Found" atau warning PaidBroadcast.PRODUCT_ID masih 0 di Output.
Membership →Paid broadcast →Donation Robux →
6

Sesuaikan roles & matikan fitur tidak dipakai

Default kit sudah punya role lengkap. Sesuaikan label, chat tag, dan privilege dengan venue kamu.

  1. Edit RoleCategories di ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig — rename role, ubah teamColor, toolFolder.
  2. Cek setiap toolFolder punya folder matching di ServerStorage/Tools/<nama folder>/ (contoh: STAFF, DONOR).
  3. Di Features (file config yang sama), set false untuk modul yang tidak dipakai.
  4. Matikan spender role yang tidak dipakai di SpenderRoles.
✓ Berhasil jika: /setrole assign role benar + tools muncul di backpack player.
Roles →Features →Tool folders →
7

Play test di Studio

Studio default pakai DataStore terisolasi (Studio_* prefix) — aman, tidak menulis data live player.

  1. Game Settings → Security → Enable Studio Access to API Services = ON.
  2. Play (F5) → test shop, ikon Broadcast (paid), /setrole, /gift, panel donasi.
  3. Test leaderboard boards di workspace tampil (kosong = normal).
  4. Admin: /testcash 1000 atau /testrobux 10 untuk cek notif + VFX (preview saja, tidak persist).
✓ Berhasil jika: tidak ada error merah di Output, GUI load normal, command admin merespons.
Studio testing →Commands →
8

Centang checklist & publish

Sebelum go-live, pastikan semua poin di kartu Deploy checklist (kanan) sudah dicentang.

  1. Review kartu checklist di kanan — klik tiap baris yang sudah selesai.
  2. Target: 11 / 11 sebelum publish.
  3. File → Publish to Roblox.
✓ Siap publish jika: checklist 11/11, showcase OFF, config & secrets terisi, sudah test di Studio.
Ke checklist →Detail deploy →

Default: Play mode di Studio pakai DataStore terisolasi (Studio_* prefix). Command persist (/addcash, /setrobux) menulis ke key terisolasi itu — bukan production player data.

Workflow QA cepat

  1. Game Settings → Security → Enable Studio Access to API Services = ON
  2. Showcase ON → /testrobux 10 → cek aura, tanpa world VFX
  3. Showcase ON → /testcash 2000 → cek aura + Nuke; board tidak berubah
  4. Butuh entry board → /addcash <player> 5000 atau seeder one-time
  5. Showcase OFF → /testcash 150000 → cek threshold production world VFX
Production keys (advanced)

Untuk debug sengaja ke live DataStore, set di ReplicatedStorage/Hazastudio_ClubKit/Shared/Constants/Config: USE_STUDIO_DATASTORE_ISOLATION = false. Kembalikan ke true setelah selesai. Command persist (/addcash, /setrobux) akan diblokir untuk Robux jika isolation off; cash manual tetap hati-hati.

Leadership: Staff · Moderation: Moderator · Content: DJ, Streamer, Influencer, LeadDance · Membership: VIP, VVIP, Supreme · Spender: Donor, Top Donor, Top Supporter.

Paid broadcast

Player membayar Robux lewat Developer Product untuk mengirim pesan broadcast ke seluruh server (ikon topbar Broadcast). Staff dengan privilege canAnnounce tetap bisa /announce gratis.

Setup Developer Product

  1. Buka Creator Dashboard → place kamu → Monetization → Developer Products.
  2. Buat product baru (mis. nama Server Broadcast) — tipe Developer Product, atur harga Robux sesuai venue.
  3. Salin Product ID (angka) dari dashboard.
  4. Paste ke ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig:
ClubKitConfig.PaidBroadcast = {
    ProductId = 3503700307, -- ganti dengan ID dari Creator Dashboard
}
Jangan campur dengan Shop

PaidBroadcast.ProductId adalah satu product terpisah — bukan Shop.Products. Server memproses receipt broadcast lewat ShopController → PaidBroadcastController.

Verifikasi

Play test → buka ikon Broadcast → ketik pesan → prompt Robux muncul dengan harga product. Jika ProductId masih 0, Output menampilkan [ConfigBootstrap] PaidBroadcast.ProductId belum diisi.

Matikan cash / Robux

Cash.Enabled = false atau Features.DonationCash = false → HTTP poll cash + tab IDR di panel donasi dimatikan. Legacy Donation.Saweria.Enabled / Features.DonationSaweria tetap didukung sebagai alias.

Robux.Enabled = false atau Features.DonationRobux = false → donasi Robux in-game dimatikan.

Test vs persist commands

/testcash dan /testrobux hanya preview (notif + aura + VFX) — tidak menulis leaderboard, DataStore, atau manual adjustment. Board tidak berubah setelah command test.

Untuk isi leaderboard sungguhan: /addcash / /setrobux (tanpa VFX) atau /donatecash (persist + VFX). Hapus manual IDR: /removecash <player>.

Alias legacy preview: /testsaweria, /testdonate → /testcash.

Seed data manual (one-time)

Untuk isi leaderboard awal (cash / robux / likes) tanpa sentuh engine Club Kit, pakai script pihak ketiga One-Time Leaderboard Seeder — lihat panduan seeder. Bukan bagian runtime kit; hapus setelah dipakai.

Live tanpa API

Board Saweria tampil "API not configured". Robux / Likes / Community kosong = "No … yet" (belum ada data), bukan error API.

Deploy rbxm

Tidak perlu Rojo atau Argon. Semua setup dilakukan langsung di Roblox Studio.

Pertama kali

  1. Buka Roblox Studio → place kamu (atau place baru)
  2. Home → Insert from File → pilih ClubKit.rbxm (nama file dari Hazastudio)
  3. Pastikan struktur kit muncul di Explorer — lihat peta path (ReplicatedStorage, ServerScriptService, StarterGui, dll.)
  4. Edit ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig dan ServerScriptService/Hazastudio_ClubKitSecrets/Secrets
  5. Play test → File → Publish to Roblox

Update kit (versi baru)

  1. Backup isi ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig + ServerScriptService/Hazastudio_ClubKitSecrets/Secrets
  2. Hapus folder engine ReplicatedStorage/Hazastudio_ClubKit dan ServerScriptService/Hazastudio_ClubKit — jangan hapus Hazastudio_ClubKitConfig / Hazastudio_ClubKitSecrets
  3. Insert rbxm versi baru
  4. Cek config buyer masih ada; restore dari backup kalau tertimpa
1 place = 1 config

Tiap place punya ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig sendiri. Duplikat place di Studio kalau mau venue berbeda.

Troubleshooting

Masalah umum saat setup dan deploy Club Kit.

Group.GROUP_ID tidak valid
Isi GroupId di ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig atau nyalakan showcase.
SHEETS_WEB_APP_URL belum dikonfigurasi
Isi ApiUrl atau pakai showcase.
Shop warning BuyId 0
Isi ID di ReplicatedStorage/Hazastudio_ClubKitConfig/ClubKitConfig → Shop.Products.
PaidBroadcast.PRODUCT_ID masih 0
Buat Developer Product broadcast di Creator Dashboard → isi PaidBroadcast.ProductId di ClubKitConfig. Lihat Paid broadcast.
Broadcast topbar tidak muncul prompt / gagal bayar
Cek ProductId benar, product aktif di dashboard, dan harga Robux sudah diset. Receipt diproses lewat shop pipeline yang sama dengan membership.
Saweria "API not configured"
Isi ApiUrl + DonationApiSecret; cek DonationCash aktif.
Robux/Likes "No … yet"
Normal — belum ada data; atau pakai one-time seeder.
Mau isi leaderboard manual untuk test
Pakai /addcash atau /setrobux (owner) — itu yang persist. /testcash dan /testrobux hanya preview VFX/notif.
Tab cash masih tampil padahal dimatikan
Cash.Enabled = false + rejoin; tab IDR disembunyikan di client.
Seeder tidak jalan
ENABLED = true, API Services ON, cek nama folder ModuleScript.
Board blank tanpa teks
Tambah LoadingOverlay di template SurfaceGui place.
Products Not Found (Robux)
Buat Developer Products atau isi RobuxProducts.
Role tidak muncul
Cek enabled = true dan alias role.
Tools tidak diberikan
Cek ServerStorage/Tools/<toolFolder>.
Owner command ditolak
Isi OwnerUserId atau group rank 255.
Warn secret kosong
Isi ServerScriptService/Hazastudio_ClubKitSecrets/Secrets atau kosongkan ApiUrl.
