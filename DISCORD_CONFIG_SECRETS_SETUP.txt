# Club Kit Рђћ Initial Setup (2/4)
ClubKitConfig Рђћ Branding, Group, Donation

**1. Branding**

```

  GameName = "Nama Club Kamu",
  WelcomeMessage = "Welcome to %s",
  Greeting = "Welcome to %s",
  LogoImage = "rbxassetid://YOUR_LOGO_ID",

```

Ganti `LogoImage` Рђћ jangan biarkan ID bawaan kit di place live.

**2. Group & Admin** (wajib)

```

  GroupId = 12345678,
  OwnerUserId = 987654321,
  OwnerGroupRank = 255,
}

AdminUserIds = {
  [111111] = true,
}
```

Cara ambil ID:
Рђб GroupId Рєњ URL halaman group
Рђб OwnerUserId Рєњ URL profil Roblox
Рђб AdminUserIds = backup admin tanpa rank group

**3. Donation** (kalau pakai Bagi-Bagi / Saweria)

```

  ApiUrl = "https://xxx.workers.dev/game/your-key",

  Robux = { Enabled = true },

}
```

Рђб `ApiUrl` kosong = board cash tidak live
Рђб Aura / WorldEffectTiers boleh default dulu

# Club Kit Рђћ Initial Setup (3/4)
ClubKitConfig Рђћ Shop, Broadcast, Features, Roles

**4. Shop membership**
Creator Dashboard Рєњ Monetization:
Рђб Game Pass = self-buy VIP / VVIP / Supreme
Рђб Developer Product = gift version

```
Shop = {
  Products = {
    Tier1 = { BuyGamePassId = 123, GiftId = 456, Price = 99 },
    Tier2 = { BuyGamePassId = 789, GiftId = 101, Price = 199 },
    Tier3 = { BuyGamePassId = 112, GiftId = 131, Price = 499 },
  },
}
```

`Price` = angka tampil di UI (bukan harga Roblox).

**5. Paid Broadcast**
Buat 1 Developer Product terpisah (bukan shop membership).

```
PaidBroadcast = {
  ProductId = 1234567890,
}
```

Kalau `0` Рєњ Output warning + broadcast berbayar mati.

**6. Membership labels** (opsional)

```
Membership = {
  Tier1 = { Label = "VIP", Enabled = true, Priority = 40 },
  Tier2 = { Label = "VVIP", Enabled = true, Priority = 50 },
  Tier3 = { Label = "Supreme", Enabled = true, Priority = 60 },
}
```

Key `Tier1` / `Tier2` / `Tier3` jangan diganti.
Kalau ganti Label, update `CommandAliases`.

**7. Features on/off**

```
Features = {
  MusicPlayer = true,
  Shop = true,
  Leaderboards = true,
  DonationCash = true,
  DonationRobux = true,
  PromptJoinCommunityOnLoad = true,
  VipOnCommunityJoin = false,
  JoinGreetings = true,
}
```

Set `false` untuk fitur yang tidak dipakai.

**8. Roles** (refine belakangan OK)
Рђб `RoleCategories` Рђћ Staff / Mod / DJ + privileges
Рђб `SystemRoles.Owner.groupRanks = { 255 }` Рђћ sesuaikan rank owner
Рђб Co-Owner default via `/setrole` (bukan group rank)
Рђб `AdminUserIds` = jalan darurat kalau rank belum map

# Club Kit Рђћ Initial Setup (4/4)
Secrets + go-live

**Path**
`ServerScriptService` Рєњ `Hazastudio_ClubKitSecrets` Рєњ `Secrets`

**Isi**

```

Secrets.GameDataApiSecret = ""
```

**Aturan Secrets**
Рђб Harus sama dengan secret di Cloudflare Worker / backend
Рђб Kosong = API donasi cash skip (board: "API not configured")
Рђб `GameDataApiSecret` kosong = Join Community fallback; normal di Studio
Рђб `GameDataApi.GameKey` di ClubKitConfig harus match key di worker
Рђб Jangan commit / kirim file Secrets berisi production key
